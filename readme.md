# Skill Level Up v1.4 - TTRPG Engine

A lightweight, web-based character management system and mechanical calculator for the custom "Skill Level up v.1.0" tabletop role-playing game. It automatically handles the complex math for combat, magic scaling, and resource tracking so players can focus on the story.

## Features
* **Live Resource Tracking:** Interactive HP, Stamina, and Mana pools that automatically calculate their maximum limits based on your Constitution, Endurance, and Mana Supply. 
* **Automated Combat Math:** Instantly calculates melee/magic damage, dodge values, hit success, and your action economy using your skill points and the current d20 roll.
* **Dynamic Unlocks:** Automatically displays passive abilities (like True Sight, Danger Sense, or Supreme Magic tiers) as you level up specific skills.
* **Zero Installation:** Built entirely with plain HTML, Tailwind CSS, and Vanilla JavaScript. No servers or databases required.

## How to Run
1. Save `index.html`, `style.css`, and `script.js` into the same folder on your computer.
2. Double-click `index.html` to open the application in any modern web browser (Chrome, Edge, Firefox, etc.).
3. Type your skill points into the input fields to see your character's stats generate in real-time.
